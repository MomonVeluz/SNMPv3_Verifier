SNMP Verification Tool

Run Instructions

1. Extract the full SNMP_Verifier_Windows.zip package to a local folder.
2. Open the extracted folder.
3. Run SNMP_Verifier.exe.

Important Notes

- Do not move SNMP_Verifier.exe away from the _internal folder.
- Do not run the executable from a PyInstaller build folder.
- Workbook verification requires an .xlsx file with a column named IP Address.
- SNMP Walk is read-only and should be used only against authorized company devices.
- SNMPv3 usernames and passwords are entered at runtime and are not saved by the application.
